# Dotfiles
This repository contains all my actively used configuration files.
## Usage:
```bash
# Git clone this repository.
$ git clone https://github.com/insidious-security/dotfiles.git

# Move the config files by choice.
$ mv .vimrc ~/
$ mv .tmux.conf ~/
```
